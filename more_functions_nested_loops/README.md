# C - More functions, more nested loops  
  
#  What is the output of the following piece of code?
```sh
int i;

for (i = 48; i < 58; i++)
{
    printf("%c", i);
}

output: 0123456789
```   
# What is the output of the following piece of code?  
```sh
int i;

i = 0;
while (i < 10)
{
    printf("%d", i % 2);
    i++;
}

output: 0101010101
```
